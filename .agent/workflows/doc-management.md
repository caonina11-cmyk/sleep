# 文档管理规则

## 项目目录
所有项目相关文件保存到：`/Users/nina/Desktop/H5页面/`

## 需要保存的文档类型
- PRD产品需求文档
- 设计规范文档
- 开发日志/更新日志
- 测试报告
- 上线检查清单
- 其他重要说明文档

## 文件命名规范
- 使用中文或英文命名均可
- 使用 `.md` 格式保存文档
- 代码文件按功能命名（如 `styles.css`, `script.js`）

## 目录结构
```
H5页面/
├── PRD.md              # 产品需求文档
├── CHANGELOG.md        # 更新日志（如需要）
├── index.html          # 着陆页
├── quiz.html           # 问卷页
├── result.html         # 结果页
├── toolkit.html        # 交付页
├── styles.css          # 全局样式
├── script.js           # 交互逻辑
└── assets/             # 静态资源
    ├── audio/          # 音频文件
    └── images/         # 图片文件
```
